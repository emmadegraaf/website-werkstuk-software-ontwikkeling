Beweging toevoegen met javascript

Om onze eigen game te programmeren, moeste we enkele nieuwe technieken leren. Ten eerste moesten we een afbeeldingen toevoegen aan het scherm. Ook moesten we deze afbeelding op een bepaalde plek neerzeten en vervolgens laten bewegen. Om dit allemaal te laten werken gebruikte we zowel html-codce, als javascript.

Als je een simple game maakt, moet je html gebruiken om ee nafbeelding toe te voegen en wat tekst om de score weer te geven. Maar html kan de afbeelding niet laten bewegen of de score aanpassen. Javascript is gemaakt om html pagina interactiever te maken.
